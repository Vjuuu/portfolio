GMB-BRIEFCASE

GMB-BRIEFCASE

About the Project:

GMB-Briefcase is a management tool created to handle multiple Google My Business profiles efficiently. Originally developed in 2016, I redesigned the platform, enhancing the user experience and introducing a fresh UI.

My Role:

  • UI/UX redesign
  • Frontend Development
  • Report Design

Key Features:

  • Citation Audit Report
  • GMB Insights Report
  • GMB Audit Report
  • Website Ranking Report
  • GEO Ranking Report
  • Redesigned Pages:
    1. Sign In & Sign Up
    2. User Onboarding
    3. Dashboard
    4. Manage Google Accounts
    5. Manage Listings
    6. Create & Display Posts (Including AI-Generated Tasks)
    7. Media Post Management
    8. Auto-Response Review Management
    9. Review Poster
    10. Manage Q&A Section
    11. Manage Subscriptions

Technologies Used:

  • HTML5
  • CSS3
  • Bootstrap 5
  • CodeIgniter 3
  • MySQL
  • HTML5
