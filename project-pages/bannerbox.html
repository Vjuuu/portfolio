About the Project:

BannerBox.in is my own project, developed for small business owners. It helps users easily download pre-designed banners for festivals, morning motivation, events, and more.

My Role:

  • Full Stack Developer (Frontend, Backend, Database)

Key Features:

  • Festival Banners
  • Motivational Posters
  • Event Templates
  • Easy Download System for user
  • Admin panel for managing Poster and Categories

Technologies Used:

  • HTML5
  • CSS3
  • Bootstrap 5
  • CodeIgniter 3
  • MySQL
  • HTML5
